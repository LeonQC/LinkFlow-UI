# LinkFlow-UI

React/Vite frontend for the LinkFlow short-link platform.

## Run locally

1. `npm install`
2. `npm run dev`

The app expects the backend API at `http://localhost:8080` by default.
Override it with `VITE_API_BASE_URL` if needed.

## Structure

- `src/`: frontend application source
- `docs/`: integration and delivery notes
- `guidelines/`: design guidance copied from the original Figma export
